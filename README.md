# OpenList-Companion
—— 专为 Alist 设计的高颜值、全自动化管理伴侣 OpenList Companion 是一款轻量级的桌面管理工具，旨在为 Alist 用户提供最直观、最便捷的操控体验。它不仅解决了命令行操作的繁琐，更通过底层逻辑的深度联动，实现了从“手动维护”到“自动托管”的跨越。
正是为你而生。欢迎提交 Issue 和 Pull Request！
✨ 核心特性
1. 🎨 极致交互与自适应布局
物理点击反馈：所有按钮均内置 Pressed 交互逻辑。点击时按钮会产生位移并变暗，提供真实的物理按压感。

窗口自适应约束：底层布局采用智能约束。无论如何拉伸窗口，按钮与功能板块始终保持完美比例，拒绝横向变形。

视觉化状态看板：服务启动瞬间，状态板会自动转为醒目的翠绿色，运行状态一目了然。

2. ⚡ 深度自动化联动 (核心功能)
初始密码秒抓取：实时监听 Alist 启动日志。一旦发现初始随机密码，软件将立即自动提取并同步至系统剪贴板，同时弹出黄色提示条。

修改密码自动生效：在软件内修改管理密码后，底层逻辑会自动重启 Alist 服务，无需人工手动干预。

一键恢复并拉起：点击导入数据恢复后，程序会自动执行：强制回收旧进程 -> 文件覆盖 -> 自动重启服务 的全闭环操作。

3. 🛡️ 完善的安全与维护
全量备份导出：一键将 Alist 数据库与配置文件压缩备份，确保数据安全。

管理凭证提取：即使忘记密码，也可通过“获取”功能瞬时解锁数据库并展示/复制当前凭证。

移动端快速接入：实时显示局域网 IP 地址，提供一键复制功能，方便手机端快速访问。

4. 📂 智能环境感应
自动感应同级目录下的 alist.exe。

支持自定义路径保存，配置文件自动锁定在程序根目录，纯绿色免安装。

🛠️ 技术栈
语言: Python 3.13+

UI 框架: PyQt5

底层处理: Subprocess / Psutil / Socket

📦 运行与下载
如果你想直接运行源码：

确保已安装依赖：pip install PyQt5 psutil requests

将脚本放置在 Alist 目录下运行即可。

注意：为了确保“获取密码”和“自动重启”功能正常运行，请务必以管理员身份运行。
