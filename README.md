# OpenList-Companion
—— 专为 Alist 设计的高颜值、全自动化管理伴侣 OpenList Companion 是一款轻量级的桌面管理工具，旨在为 Alist 用户提供最直观、最便捷的操控体验。它不仅解决了命令行操作的繁琐，更通过底层逻辑的深度联动，实现了从“手动维护”到“自动托管”的跨越。
✨ 核心特性
 🎨 极致的交互反馈与自适应布局
物理点击感：全按钮加入 Pressed 交互逻辑，模拟真实的物理按压反馈，操作不再“软绵绵”。

窗口自适应：重构底层布局约束，无论窗口如何拉伸，按钮与元素始终保持完美比例，拒绝变形。

状态视觉化：采用高对比度状态板，服务开启瞬间转为醒目的“翠绿色”，运行状态一眼便知。

⚡ 底层自动化联动（新功能！）
初始密码秒抓取：针对新用户痛点，软件会自动监听控制台日志。一旦发现初始随机密码，立即自动提取并同步至系统剪贴板，同时弹出黄色高亮提示。

修改密码自动生效：在软件内修改管理密码后，底层逻辑会自动重启服务，无需人工手动干预。

一键恢复并拉起：点击导入数据恢复后，程序会自动执行：强制回收进程 -> 文件覆盖 -> 自动重启服务 的全闭环操作。

🛡️ 安全与维护
数据全量备份：支持一键导出 Alist 完整数据库与配置，确保数据万无一无。

管理凭证提取：即使忘记密码，也可通过“获取”功能瞬时解锁数据库并提取当前凭证。

手机端无缝访问：实时显示局域网 IP 链接，一键复制，方便移动端快速接入。

 📂 路径自感应
智能感应同级目录下的 alist.exe，支持自定义路径保存，配置自动锁定在程序根目录，绿色免安装。

🛠️ 技术栈
Language: Python 3.13+

UI Framework: PyQt5

Networking: Socket / Requests

Process Management: Psutil

📦 打包与运行
本项目专门优化了源码运行环境与打包兼容性：

调试模式：直接在编辑器运行，已移除自动提权导致的闪退逻辑，方便实时预览效果。

打包建议：使用 PyInstaller 进行 --onefile --windowed 打包，内置资源路径已自动处理。
